**Media Player**

Media Player is a MERN stack application that enables users to add, delete, categorise songs into different categorise and provide the feature of monitoring the history of tracks.

**Tech Stack used**: 
1) MongoDB
2) React JS
3) Express JS
4) Node JS
5) Bootstrap
6) CSS3
7) Axios

**Live Link** : https://media-frontend-six.vercel.app/

**Link of Backend** : https://github.com/amalsunny-cloud/media-server
